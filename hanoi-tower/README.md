# 🗼 ハノイの塔 - 小学生向けWebアプリ

小学生が楽しく論理的思考を学べるハノイの塔のインタラクティブなWebアプリです。

## 📋 特徴

- **3段階の難易度**: かんたん（3個）→ ふつう（4個）→ むずかしい（5個）
- **ドラッグ&ドロップ操作**: 直感的でわかりやすい操作方法
- **ルール自動チェック**: ルール違反は自動的に警告
- **移動回数追跡**: 最小移動回数との比較で上達が可認識できる
- **わかりやすい説明**: ハノイの塔の遊び方とコツを掲載
- **カラフルなデザイン**: 子どもたちが使いやすい見た目

## 🎮 遊び方

### 基本ルール
1. **大きなリングの上に小さなリングを置くことはできません**
2. **1回に1つのリングだけ移動できます**
3. **すべてのリングを左の柱から右の柱に移動させます**

### オススメの進め方
1. スタートは「かんたん（3個）」で遊ぶ
2. パターンを理解したら「ふつう（4個）」に挑戦
3. 最後に「むずかしい（5個）」に挑戦

## 📊 最小移動回数

- リング3個: 7回
- リング4個: 15回
- リング5個: 31回

この数字より少ない回数でクリアできたら素晴らしい！

## 📖 ハノイの塔について

ハノイの塔は、論理的思考と問題解決のスキルを養う古典的なパズルです。

**教育的価値:**
- 論理的思考の発達
- 問題解決能力の向上
- パターン認識力の強化
- 数学的概念の理解（指数関数など）

## 📁 ファイル構成

```
hanoi-tower/
├── index.html      - メインのHTMLファイル
├── style.css       - スタイルシート
└── script.js       - ゲームロジック
```

## 🚀 使い方

1. `index.html`をブラウザで開く
2. 難易度を選択
3. ドラッグ&ドロップでリングを移動
4. すべてのリングを右の柱に移動させてクリア！

## 💡 学習効果

このアプリを通じて、子どもたちは以下を学べます：

- **ステップバイステップの問題解決**
- **複雑な問題を小さな問題に分割すること**
- **ルールに従うこと**
- **試行錯誤を通じた学習**
- **成功体験の獲得**

## 🎨 カスタマイズ方法

### リング数を変更したい場合
`script.js`のカラー配列を拡張し、`startGame()`の引数を変更してください。

### デザインを変更したい場合
`style.css`のスタイルを修正してください。
